# Frontle

> Experience the easiest multi-platform SPA development in the world!

## Install

```shell
npm install -g cordova
npm install -g @frontle/frontle-cli
```

## Creating a new Frontle project

```shell
frontle create myApp
cd myApp
frontle install
cordova platform add browser
cordova run browser
```
